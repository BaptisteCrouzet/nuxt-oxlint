## Quick Setup

Install the module to your Nuxt application with one command:

```bash
npx nuxt module add nuxt-oxlint
```

Or with pnpm
```bash
pnpm dlx nuxt module add nuxt-oxlint
```

That's it! You can now use My Module in your Nuxt app ✨

<details>
  <summary>Local development</summary>
  
  ```bash
  # Install dependencies
  pnpm install
  
  # Generate type stubs
  pnpmdev:prepare
  
  # Develop with the playground
  pnpmdev
  
  # Build the playground
  pnpmdev:build
  
  # Run ESLint
  pnpm lint
  
  # Run Vitest
  pnpm test
  ```
